## Overview

The `Tiempo` class is an activity that displays current weather information for Huelva, Spain. It fetches data from the OpenWeatherMap API using Volley for network requests and displays temperature, weather conditions, humidity, wind speed, and other meteorological data.

<Warning>
The API key is currently hardcoded in the source. Consider moving it to `local.properties` and accessing via `BuildConfig` to prevent exposure in version control.
</Warning>

## API Integration

### Request Construction

<CodeGroup>

```java Building the API URL
String urlCompleta = URL_BASE + "?q=Huelva,ES&appid=" + API_KEY;
// Results in: http://api.openweathermap.org/data/2.5/weather?q=Huelva,ES&appid=c0d138083fb0634d6d5cd36adafc9a77
```

```java Volley Request
StringRequest stringRequest = new StringRequest(Request.Method.POST, urlCompleta,
    response -> {
        // Success callback
        try {
            JSONObject jsonResponse = new JSONObject(response);
            // Parse and display data
        } catch (JSONException e) {
            e.printStackTrace();
        }
    },
    error -> {
        // Error callback
        Toast.makeText(getApplicationContext(), 
            error.toString().trim(), 
            Toast.LENGTH_SHORT).show();
    }
);

RequestQueue requestQueue = Volley.newRequestQueue(getApplicationContext());
requestQueue.add(stringRequest);
```

</CodeGroup>

### JSON Response Parsing

The activity parses the following data from the API response:

<Tabs>
  <Tab title="Weather Description">
    ```java
    JSONArray jsonArray = jsonResponse.getJSONArray("weather");
    JSONObject jsonObjectWeather = jsonArray.getJSONObject(0);
    String descripcion = jsonObjectWeather.getString("description");
    traducirDescripcion(descripcion);
    ```
  </Tab>
  
  <Tab title="Temperature Data">
    ```java
    JSONObject jsonObjectMain = jsonResponse.getJSONObject("main");
    double temp = jsonObjectMain.getDouble("temp") - 273.15; // Kelvin to Celsius
    double sensacion = jsonObjectMain.getDouble("feels_like") - 273.15;
    float presion = jsonObjectMain.getInt("pressure");
    int humedad = jsonObjectMain.getInt("humidity");
    
    tvTemperatura.setText(formatoDecimal.format(temp) + "ºC");
    tvSensacion.setText("Sensacion térmica: " + formatoDecimal.format(sensacion) + "ºC");
    tvHumedad.setText("Humedad: " + humedad + "%");
    tvPresion.setText("Presión atmosférica: " + presion + " hPa");
    ```
  </Tab>
  
  <Tab title="Wind & Clouds">
    ```java
    JSONObject jsonObjectWind = jsonResponse.getJSONObject("wind");
    String viento = jsonObjectWind.getString("speed");
    
    JSONObject jsonObjectClouds = jsonResponse.getJSONObject("clouds");
    String nubes = jsonObjectClouds.getString("all");
    
    tvViento.setText("Velocidad del viento: " + viento + "m/s (metros por segundo)");
    tvNubes.setText("Cantidad de nubes: " + nubes + "%");
    ```
  </Tab>
</Tabs>

## Error Handling

<Expandable title="Network Errors">
  Network errors (no internet, timeout, etc.) are displayed via Toast:
  
  ```java
  error -> Toast.makeText(getApplicationContext(), 
      error.toString().trim(), 
      Toast.LENGTH_SHORT).show()
  ```
  
  **Common errors:**
  - `NoConnectionError`: No internet connection
  - `TimeoutError`: Request took too long
  - `ServerError`: API server error (5xx)
  - `AuthFailureError`: Invalid API key
</Expandable>

<Expandable title="JSON Parsing Errors">
  JSON parsing errors are caught and logged:
  
  ```java
  try {
      JSONObject jsonResponse = new JSONObject(response);
      // ... parsing code ...
  } catch (JSONException e) {
      e.printStackTrace();
  }
  ```
  
  Consider adding user-facing error handling:
  ```java
  catch (JSONException e) {
      e.printStackTrace();
      Toast.makeText(this, "Error processing weather data", Toast.LENGTH_SHORT).show();
  }
  ```
</Expandable>

## Temperature Conversion

The API returns temperatures in Kelvin. The activity converts to Celsius:

```java
double temp = jsonObjectMain.getDouble("temp") - 273.15;
double sensacion = jsonObjectMain.getDouble("feels_like") - 273.15;
```

**Conversion formula:**
```
Celsius = Kelvin - 273.15
```

**Example:**
- API returns: `298.15 K`
- Displayed: `25.0ºC`

## Security Recommendations

<Warning>
**API Key Exposure**

The current implementation exposes the API key in source code. Follow these steps to secure it:

1. **Add to local.properties:**
   ```properties
   OPENWEATHER_API_KEY=c0d138083fb0634d6d5cd36adafc9a77
   ```

2. **Update build.gradle:**
   ```gradle
   android {
       defaultConfig {
           Properties properties = new Properties()
           properties.load(project.rootProject.file("local.properties").newDataInputStream())
           buildConfigField "String", "OPENWEATHER_API_KEY", "\"${properties.getProperty('OPENWEATHER_API_KEY')}\""
       }
   }
   ```

3. **Update Tiempo.java:**
   ```java
   private static final String API_KEY = BuildConfig.OPENWEATHER_API_KEY;
   ```

4. **Add to .gitignore:**
   ```
   local.properties
   ```
</Warning>
