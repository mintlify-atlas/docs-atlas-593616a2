## Overview

The `Elementos` class is a Java data model class that represents an encyclopedia entry in the HuelvaPedia app. It stores all information related to a topic, including name, photo, description, location, contact details, and additional metadata.

<Note>
This class is designed to work seamlessly with Firebase Realtime Database for data persistence and retrieval.
</Note>

## Class Declaration

```java
package com.example.huelvapedia;
```

<ResponseField name="ubicacion2" type="String">

</ResponseField>

<ResponseField name="telefono" type="String">
Contact phone number for the entry (if applicable).
</ResponseField>

<ResponseField name="ultimo" type="Boolean">
Flag indicating whether this is the last/final element in a navigation hierarchy. When `true`, the entry displays action buttons instead of allowing further navigation.
</ResponseField>

<ResponseField name="enlace" type="String">
External web link or reference URL for additional information.
</ResponseField>

## Usage Example

<CodeGroup>

```java Creating and populating an Elementos object
// Create a new encyclopedia entry
Elementos playa = new Elementos();
playa.setNombre("Playa de Punta Umbría");
playa.setFoto("https://example.com/punta-umbria.jpg");
playa.setDescripcion("Una de las playas más populares de Huelva");
playa.setUbicacion1("https://maps.google.com/?q=");
playa.setUbicacion2("Playa Punta Umbría, Huelva");
playa.setTelefono("+34959123456");
playa.setUltimo(true);
playa.setEnlace("https://www.puntaumbria.es/");
```

```java Using the parameterized constructor
Elementos monumento = new Elementos(
    "Monumento a la Fe Descubridora",
    "https://example.com/monumento.jpg",
    "Estatua en honor a Cristóbal Colón",
    "https://maps.google.com/?q=",
    "Monumento Fe Descubridora, Huelva",
    null, // No phone number
    true,
    "https://es.wikipedia.org/wiki/Monumento_a_la_Fe_Descubridora"
);
```

```java Retrieving data from Firebase
reference.addValueEventListener(new ValueEventListener() {
    @Override
    public void onDataChange(@NonNull DataSnapshot snapshot) {
        for (DataSnapshot dataSnapshot : snapshot.getChildren()) {
            Elementos elemento = dataSnapshot.getValue(Elementos.class);
            Log.d("HuelvaPedia", elemento.getNombre());
        }
    }
    
    @Override
    public void onCancelled(@NonNull DatabaseError error) {
        // Handle error
    }
});
```
