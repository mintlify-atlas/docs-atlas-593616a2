## Overview

The `Adaptador` class is a custom `RecyclerView.Adapter` that manages the display of encyclopedia entries (`Elementos` objects) in the HuelvaPedia app. It handles view binding, click events, and conditional UI display based on entry properties.

<Note>
This adapter implements `View.OnClickListener` to handle item click events and supports both navigation items and actionable items with buttons.
</Note>

## Class Declaration

### onBindViewHolder

```java
@Override
public void onBindViewHolder(@NonNull MiViewHolder holder, int position)
```

Binds data from an `Elementos` object to the ViewHolder at the specified position. This method handles all the UI logic including:
- Setting text and images
- Showing/hiding action buttons
- Configuring click listeners based on entry type

<ParamField path="holder" type="MiViewHolder">
The ViewHolder to bind data to
</ParamField>

<ParamField path="position" type="int">
The position of the item in the data set
</ParamField>

**Behavior:**

<Tabs>
  <Tab title="Navigation Items (ultimo = false)">
    When `elemento.getUltimo()` returns `false`:
    - Hides all action buttons (location, phone, link)
    - Sets up click listener to navigate to `Secundaria` activity
    - Passes the entry name as an intent extra
  </Tab>
  
  <Tab title="Action Items (ultimo = true)">
    When `elemento.getUltimo()` returns `true`:
    - Displays action buttons based on available data
    - **Location button**: Opens map with the location if `ubicacion1` and `ubicacion2` are set
    - **Phone button**: Opens dialer if `telefono` is set
    - **Link button**: Opens web browser if `enlace` is set
    - Hides buttons for unavailable actions
  </Tab>
</Tabs>

**Example:**
```java
// This method is called automatically by RecyclerView
// when an item needs to be displayed or updated
```

## Inner Class: MiViewHolder

The `MiViewHolder` class is a static inner class that holds references to the views for each item in the RecyclerView.

```java

```

## Usage Examples

<CodeGroup>

```java Basic adapter setup
// In your Activity or Fragment
RecyclerView recyclerView = findViewById(R.id.recicler);
recyclerView.setLayoutManager(new LinearLayoutManager(this));

ArrayList<Elementos> listaElementos = new ArrayList<>();
Adaptador adaptador = new Adaptador(this, listaElementos);
recyclerView.setAdapter(adaptador);
```

```java With Firebase data loading
reference.addValueEventListener(new ValueEventListener() {
    @Override
    public void onDataChange(@NonNull DataSnapshot snapshot) {
        listaElementos.clear();
        for (DataSnapshot dataSnapshot : snapshot.getChildren()) {
            Elementos elemento = dataSnapshot.getValue(Elementos.class);
            listaElementos.add(elemento);
        }
        adaptador = new Adaptador(MainActivity.this, listaElementos);
        recyclerView.setAdapter(adaptador);
    }
    
    @Override
    public void onCancelled(@NonNull DatabaseError error) {
        Toast.makeText(MainActivity.this, "Error loading data", Toast.LENGTH_SHORT).show();
    }
});
```

```java Custom click listener
Adaptador adaptador = new Adaptador(this, listaElementos);
adaptador.setOnClickListener(v -> {
    int position = recyclerView.getChildAdapterPosition(v);
    if (position != RecyclerView.NO_POSITION) {
        Elementos elemento = listaElementos.get(position);
        Log.d("HuelvaPedia", "Clicked: " + elemento.getNombre());
    }
});
recyclerView.setAdapter(adaptador);
```

</CodeGroup>

## Action Button Behavior

The adapter handles three types of action buttons for final entries:

<Expandable title="Location Button (Map)">
  When both `ubicacion1` and `ubicacion2` are available:
  ```java
  // Opens map application with the location
  Intent intent = new Intent(Intent.ACTION_VIEW);
  intent.setData(Uri.parse(elemento.getUbicacion1() + Uri.encode(elemento.getUbicacion2())));
  Intent chooser = Intent.createChooser(intent, "Launch Maps");
  v.getContext().startActivity(chooser);
  ```
</Expandable>

<Expandable title="Phone Button (Dialer)">
  When `telefono` is available:
  ```java
  // Opens phone dialer with the number
  Intent intent = new Intent(Intent.ACTION_DIAL);
  intent.setData(Uri.parse("tel:" + elemento.getTelefono()));
  v.getContext().startActivity(intent);
  ```
</Expandable>

<Expandable title="Link Button (Web Browser)">
  When `enlace` is available:
  ```java
  // Opens web browser with the URL
  Uri link = Uri.parse(elemento.getEnlace());
  Intent intent = new Intent(Intent.ACTION_VIEW, link);
  v.getContext().startActivity(intent);
  ```
</Expandable>
