## What is HuelvaPedia?

HuelvaPedia is a native Android encyclopedia application that showcases the cultural, historical, and geographical richness of Huelva, Spain. Built with Java and powered by Firebase Realtime Database, it provides users with up-to-date information about local gastronomy, historical landmarks, points of interest, and real-time weather data.

## Tech Stack

HuelvaPedia is built with modern Android development tools and services:

- **Language**: Java
- **Minimum SDK**: API 27 (Android 8.1 Oreo)
- **Target SDK**: API 30 (Android 11)
- **Backend**: Firebase Realtime Database
- **Networking**: Android Volley for HTTP requests
- **Image Loading**: Picasso library
- **UI Framework**: AndroidX with Material Design components

## Prerequisites

To work with HuelvaPedia, you'll need:

- Android Studio (Electric Eel or newer)
- Android SDK (API 30+)
- Java JDK 11 or higher
- An active internet connection for Firebase and API access
- A Firebase project with Realtime Database configured

<Note>
  The app requires an internet connection to function properly, as all content is fetched dynamically from the cloud.
</Note>
